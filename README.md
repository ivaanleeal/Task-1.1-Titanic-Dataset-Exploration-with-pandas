# Explicación de la Tarea :bookmark_tabs:

En esta tarea se trabajará con diferentes métodos de la librería pandas aplicados a un archivo CSV, en este caso el dataset del Titanic.
El objetivo principal es entender las funciones básicas de inspección, exploración y análisis de datos, entendiendo:

- Qué información proporciona cada método.

- En qué situaciones se utiliza.

- Cómo interpretar sus resultados.

- Cómo estos métodos ayudan a preparar los datos para análisis posteriores.

Este ejercicio sirve como introducción de datasets con pandas.

---

# Ejecución de la Tarea: :paperclip:
Para cada método empleado se creó un apartado independiente que incluye:

- Explicación del método: qué hace, para qué sirve y cuál es su práctica.

- Ejemplo en código: demostración aplicada al dataset del Titanic.

- Resultado obtenido: salida generada por pandas al ejecutar dicho método.

- Interpretación breve: comentario de lo que se observa o aprende a partir de ese resultado.

Este formato nos permite ver paso a paso cómo se utiliza cada método en un dataset.
